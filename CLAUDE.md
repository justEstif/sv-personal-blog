## Project Overview

This is a **SvelteKit-based personal blog** project using modern web technologies:

- **Framework**: SvelteKit 2.47+ (full-stack Svelte framework)
- **Styling**: Tailwind CSS v4 with DaisyUI components
- **Language**: TypeScript (strict mode)
- **Build Tool**: Vite
- **Package Manager**: pnpm
- **Environment/Dev Tools**: mise
